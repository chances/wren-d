# wren-d

[![DUB Package](https://img.shields.io/dub/v/wren-d.svg)](https://code.dlang.org/packages/wren-d)

D bindings to the [wren](http://wren.io/) scripting language.

## Usage

```json
"dependencies": {
    "wren-d": "0.6.0"
}
```

See Wren's [Embedding](https://wren.io/embedding/) instructions.
